# 财经简报排版规范（QQ/微信推送）

适用于今日参考、盘中速递、收盘复盘等财经定时推送。编排目标是：
**30秒扫完、重点突出、可直接用于操作决策。**

## 核心原则

1. **表格优先**：所有数据组合（指数、基金涨跌、分组对比）必须用 markdown 表格，不要纯文字堆叠。
2. **颜色遵循 A股惯例**：🔴=涨 🟢=跌 🟡=持平（与美股相反，用户会纠正你）。
3. **克制字数**：总字数严格控制在 1200 以内（QQ 和微信过长会折叠）。
4. **每条数据→可操作**：不能推导出操作建议的数据不展示。

## 表格格式

所有 markdown 表格必须有表头行和对齐线，且用 emoji 标注涨跌：

### ✅ 正确示例

```
📈 **大盘实时**
| 指数 | 点位 | 涨跌 |
|------|------|------|
| 上证指数 | 4082.13 | 🔴 +0.20% |
| 创业板指 | 4345.84 | 🔴 +3.06% |
| 黄金ETF | 8.618 | 🟢 -2.33% |
```

```
💰 **持仓分组**
| 组别 | 涨跌 | 支数 |
|------|------|------|
| 科技/AI | 🔴 +2.82% | 10支 |
| 黄金 | 🟢 -2.38% | 2支 |
```

### ❌ 错误示例（用户明确反对的格式）

```
📈 上午行情
🔴 上证指数 4082.13 +0.20%
🔴 创业板指 4345.84 +3.06%
```

纯文字罗列没有视觉结构，用户反馈"数据堆在一起，没有表格结构"。

```

**🌙 隔夜外盘**  (周二至周五)
| 指数 | 点位 | 涨跌 |
|------|------|------|

**📈 A股昨收**
| 指数 | 点位 | 涨跌 |
|------|------|------|

**💰 持仓分组**
| 组别 | 涨跌 | 支数 |
|------|------|------|

**📰 博主信号**
[博主名] "引语" → 解读

**📌 今日关注**
- 核心操作建议(2-3条)
```

## 博主信号展示规则

| 博主 | 展示方式 |
|------|---------|
| 唐史主任司马迁 | ✅ 展示，黑话标注AI解读 |
| 小浣熊1230 | ✅ 展示，直接引用 |
| IT精英带你养基 | ❌ **不展示**（用户已认定无信号价值） |

- 每条博主信号控制在 1~2 行
- 黑话解读用 `→` 分隔：`"关键引语" → 解读`
- 评论区的信号用 `[评论区]` 前缀标注

## 日期标注规则

- 周二至周五：隔夜外盘数据新鲜，正常写"🌙 隔夜外盘"
- 周一：外盘数据来自上周五（已过3天），写"🌙 上周五外盘收盘"，并用 web_search 补充周末宏观事件写进"📰 周末大事"
