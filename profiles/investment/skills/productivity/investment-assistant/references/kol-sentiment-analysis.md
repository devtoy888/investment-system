# KOL 赛道情绪分析系统（2026-07-14 新增）

> 基于关键词匹配的 KOL 微博赛道分类 + 方向判断，从"罗列15条"升级为"情绪共识+操作建议"

## 架构

```
微博采集 (15条/博主)
     │
     ▼
赛道关键词匹配 → 方向关键词匹配 → 每条标注 sector + direction
     │
     ▼
按博主汇总 → 按赛道汇总 → 生成 KOL共识
     │
     ▼
存入 _kol_summary.txt  →  卡片推送
     │
     ▼
存入 _kol_consensus.txt →  共识小结卡片
```

### DIRECTION_BULLISH / BEARISH（方向判断）

```python
DIRECTION_BULLISH = ['加仓','补仓','右侧','底部','触底','反弹','抄底','建仓','看多','看好',
                     '确定性','吃肉','弹性','格局','扫货','拿住','定投','不慌','机会']
DIRECTION_BEARISH = ['减仓','清仓','风险','警惕','回调','泡沫','过热','出货','砸盘','逃顶',
                     '止损','谨慎','离场','回避','空仓','别追','小心','等待','观望']
```

判断逻辑：一条微博中提到某赛道 + 含任意看多关键词 → `bullish`
          一条微博中提到某赛道 + 含任意看空关键词 → `bearish`
          一条微博中提到某赛道 + 无方向关键词 → `neutral`

**注意**：同一赛道可同时触发 bullish 和 bearish（如"加仓科技但警惕风险"）。当前先判断 bullish 再判断 bearish，后者覆盖前者。如需更精细处理可改为看多/看空分别计数。

## 输出文件格式

### `_kol_summary.txt`（KOL 详细列表）

```
─── 唐史主任司马迁 ───
📊 **赛道情绪汇总**
  🔴 市场整体: 📈2 📉0 ➖6 （净方向: +2）
  🔴 科技/AI: 📈1 📉0 ➖1 （净方向: +1）

📋 最新 15 条:
  📝 市场在形成共识之前，还会处于高波的状态...
  🏷️ ➖市场整体
  🔄61 💬282 ❤️3111

  📝 量化是客观存在的...
  🏷️ ➖市场整体
  🔄175 💬827 ❤️10042
```

每行格式：
- `📊 **赛道情绪汇总**` — 汇总段，按赛道统计看多/看空/中性条数 + 净方向
- `📝 ...` — 微博文本摘要（前 120 字）
- `🏷️ ...` — 赛道标签，带方向 emoji（📈=bullish, 📉=bearish, ➖=neutral）
- `🔄💬❤️` — 互动数据

### `_kol_consensus.txt`（共识汇总，供卡片直接展示）

```
📊 **KOL 赛道共识**
共分析 25 条信号，主要赛道方向：
  🟢 科技/AI: 📈1 📉8 ⏸️ **积极看空**
  🟡 市场整体: 📈2 📉2 ➖ **中性/分歧**
  🟡 黄金: 📈0 📉0 ➖ **中性/分歧**
  **整体**: 🟢 **整体偏空** — KOL共识偏向谨慎，注意控制仓位
```

操作建议映射：

| 净方向 | 建议 |
|:-----:|:----|
| net ≥ 3 | ▶️ **积极看多** |
| net ≥ 1 | 📈 **偏多** |
| net ≤ -3 | ⏸️ **积极看空** |
| net ≤ -1 | 📉 **偏空** |
| 其他 | ➖ **中性/分歧** |

整体判断：

| 条件 | 结论 |
|:---|:----|
| 净多 ≥ 5 | 🔴 **整体偏多** — 可关注加仓信号 |
| 净空 ≥ 5 | 🟢 **整体偏空** — 注意控制仓位 |
| 净差 < 2 | 🟡 **整体中性** — 等待明确信号 |
| 其他 | 🟠 **轻度偏多/偏空** |

- `fund_tools.py::SIGNALS` 字典 — 黑话解读（独立于赛道分类，并行使用）

## 局限性

1. **关键词精度有限** — "黄金"可能指贵金属观点，也可能只是顺带提及
2. **方向词冲突** — 一条微博可能同时看多和看空（如"加仓科技但注意风险"）
3. **上下文缺失** — 不能区分引用他人观点和自身观点
4. **无LLM增强** — 纯规则匹配，可考虑接入LLM做语义分析提升精度
