# 操作建议系统设计 (维度C)

## 概述

2026-07-01 上线。把"看到数据→决定操作"的决策过程从AI直觉判断转为结构化评分+规则引擎。

## 数据流

```
收盘复盘的 _yesterday_snapshot.json
         │
         ▼
今日参考 pre-script ───→ _operation_plan.txt ──→ AI prompt 原样输出
         │
         ├─ check_rebalance(funds)      → 权重偏离检测
         ├─ score_group_action(各组的)   → 多空评分 → 操作建议
         │    趋势(最新数据=昨日行情, 开盘前无趋势)
         │    板块动量(引用上午数据回调昨日场景)
         │    KOL信号(从博文提取看多/看空)
         │
收盘复盘 pre-script ───→ _operation_eval.txt ──→ AI prompt 原样输出
         │
         ├─ record_group_trend(funds)   → 追加到 _group_trends.jsonl
         ├─ get_group_trend(group,days) → 读取N天趋势
         ├─ score_group_action(...)     → 趋势+信号+动量综合评分
         └─ check_rebalance(funds)      → 收盘再平衡检查
```

## 评分规则

### score_group_action(group_name, quotes, sectors, kol_signals, trend_data)

| 维度 | 数据源 | 权重权重 |
|:-----|:-------|:--------:|
| 趋势得分 | get_group_trend() 最近3日均涨跌 | ±2 |
| 板块动量 | quotes[科创50/黄金ETF市场价/…] | ±2 |
| KOL信号 | kol_signals 中匹配该组的关键词 | ±1 |
| **总分** | | **-5 ~ +5** |

### 综合判定

| 得分 | 建议 | 信号emoji |
|:---:|:----|:---------:|
| ≥ +3 | **增持** | 🟢 |
| +1 ~ +2 | **关注** | 🟢 |
| 0 | **持有** | 🟡 |
| -1 ~ -2 | **观望** | 🔴 |
| ≤ -3 | **减持** | 🔴 |

## 再平衡检查

`check_rebalance(funds)` 单日涨跌造成的估算权重偏移：
```
new_weight = cfg['weight'] * (1 + avg_change/100)
if new_weight > target_max + trigger → ⚠️ 超上限，建议部分止盈
if new_weight < target_min - trigger → 💡 低于下限，逢低补仓
```

## 趋势记录文件格式

`_group_trends.jsonl` (append-only, 每日收盘追加一行):
```jsonl
{"_date": "2026-06-30", "科技/AI": -1.5, "黄金": 0.3, "资源/周期": -0.8, "新能源": -1.1, "通航": 0.6}
```

## 操作建议文件格式

### _operation_plan.txt (今日参考)

```
📋 **今日操作参考**
| 组别 | 方向 | 建议 | 理由 |
|:----|:----:|:----|:-----|
| 科技/AI | 🟡 | 持有 | 无明显信号 |
| 黄金 | 🔴 | 观望 | 黄金ETF市场价跌-1.02% |

✅ 再平衡检查：各组占比均在目标范围内，无需调整
```

### _operation_eval.txt (收盘复盘)

```
📋 **操作评估**
| 组别 | 信号 | 今日操作建议 | 收盘表现 | 评估 | 明日方向 |
|:----|:----:|:-----------:|:--------:|:----:|:--------:|
| 科技/AI | 🔴 | 观望 | 📉 -0.69% | ✅ 正确 | 观望/减持 |

📊 **本周分组趋势**
| 分组 | 走势 | 本周幅度 |
|:----|:---:|:--------:|
| 黄金 | ▁▂▇▇█ | -1.1% → +1.4% 📈 持续上行 |
| 科技/AI | ▇▆▁▁█ | -5.3% → -0.1% 📈 跌幅收窄 |

✅ 再平衡检查：各组占比均在目标范围内
```

**注意：** 趋势速览已从纯文本改为 sparkline 走势图表格。`closing_review.py` 自动生成，不需要 AI 手动排版。sparkline 使用 Unicode 8级色阶字符（`▁▂▃▄▅▆▇█`），数值归一化后映射。

## Pitfalls

1. **趋势数据在开盘前为空** — 今日参考的评分只有板块动量+KOL信号，无趋势维度。收盘复评才有完整3维评分。
2. **趋势文件重复记录** — 测试时多次运行closing_review.py会导致同一天重复记录。生产环境无此问题(每天只跑一次收盘复盘)。
3. **收盘复盘评估"正确"判断规则** — 操作建议为"增持/关注"且收盘涨 → ✅；"减持/观望"且收盘跌 → ✅；"持有" → ➖中性。这是粗略判断，趋势走强时即使"持有"也可能是正确决策。
4. **权重是估算值** — PORTFOLIO_WEIGHTS 基于基金数量等权估算，不是用户实际投入金额。精确权重需用户提供。
5. **盘中速递未集成操作建议** — 当前仅在今日参考和收盘复盘推送中有操作板块，盘中速递待后续。
