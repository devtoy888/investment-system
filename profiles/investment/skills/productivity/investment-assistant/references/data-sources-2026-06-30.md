# 数据源端点汇总 — 2026-06-30 新增

## 行业板块（腾讯批量查询）

**URL:** `https://qt.gtimg.cn/q={逗号分隔的代码}`
**特点:** 一次HTTP查询返回所有代码数据，JSONP格式多行拼接
**字段:**
- [1] 名称 [3] 现价 [4] 昨收 [5] 今开
- [6] 成交量 [7] 成交额(ETF才有，指数为0)
- [31] 涨跌额 [32] 涨跌幅% [33] 最高 [34] 最低
- [35] `价格/成交量/成交额` (三段式，指数和ETF都可用)
- [37] 换手率 [39] 市盈率 [45] 总市值 [44] 流通市值
- [30] 时间戳 (如 `20260630120500`)

**代码前缀规则:** 6xx/9xx→`sh`, 8xx→`bj`, 其他→`sz`
**批量返回:** 多行以 `;` 拼接，field[2]是纯数字代码(无sh/sz前缀)

## 涨跌家数（东财 push2 API）

**URL:** `https://push2.eastmoney.com/api/qt/stock/get?secid=1.000001&fields=f57,f58,f169,f170,f171`
**Header:** Referer=https://quote.eastmoney.com, User-Agent=Mozilla/5.0
**返回:**
- f169 = 上涨家数
- f170 = 下跌家数
- f171 = 平盘家数
- f167 = 涨停家数(可能)
- f168 = 跌停家数(可能)

## 两市成交额（腾讯指数 field[35]）

**URL:** `https://qt.gtimg.cn/q=sh000002` (A股指数) + `https://qt.gtimg.cn/q=sz399001` (深证成指)
**解析:** field[35] = `价格/成交量/成交额(元)` 三段式，取第三段
**合计:** 上证(A股指数) + 深证(深证成指) = 两市总成交额(元)
**屏蔽:** 非交易时段返回 0，预采集脚本跳过

## 北向资金（同花顺 hexin API）

**URL:** `https://data.hexin.cn/market/hsgtApi/method/dayChart/`
**Header:** Host=data.hexin.cn, Referer=https://data.hexin.cn/, User-Agent=Mozilla/5.0
**返回:** 262个分钟点 (09:10–15:00)
- `time[]` = 时间标签数组
- `hgt[]` = 沪股通累计净买入(亿元)
- `sgt[]` = 深股通累计净买入(亿元)
**取值:** 取最后一个非None有效值。交易日收盘后累计值冻结不再更新。
**重试:** API偶有中断，函数内置1次重试(1秒间隔)。
