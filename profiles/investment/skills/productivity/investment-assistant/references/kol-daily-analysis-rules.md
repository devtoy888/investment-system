# KOL 日报分析规则（2026-07-15 更新）

## 核心原则

| 旧行为 | 新行为 |
|--------|--------|
| 每条微博都跑 `interpret_weibo()` | 仅今日信号帖才做深度解读 |
| 近几日微博与今日等同处理 | 近几日仅做赛道趋势统计（看一致性） |
| 显示互动数（转评赞） | 全部移除，纯文本输出 |
| 每帖都分析评论区 | 仅主任信号帖拉评论，只提取主任回复 |
| 黑话破译大段输出 | 精简，只对有明确信号的帖子 |
| 全部采信KOL说法 | 增加事实核查，对照行情数据验证 |

## 采集分类策略

```
get_user_weibos(uid, count=15) 
→ 按日期分类
  ├── 今日帖子 → 赛道分析 + 信号词检测 → 黑话破译 + 事实核查
  └── 历史帖子 → 仅赛道分析（不计入深度输出）
```

### 日期判断

Weibo `created_at` 格式: `"Sat Dec 28 20:44:17 +0800 2024"`
```python
def _is_today_post(post):
    d = datetime.strptime(post['created_at'], '%a %b %d %H:%M:%S %z %Y').date()
    return d == date.today()
```

### 信号词触发

只有帖子包含以下词汇才做 `interpret_weibo()`:
```python
SIGNAL_TRIGGERS = ['加仓','减仓','清仓','抄底','反弹','底部','泡沫','风险',
                   'ICU','KTV','IPO','放量','缩量','格局','右侧','机会','警惕',
                   '止损','触底','泡沫','过热','扫货','建仓','离场','逃顶']
```

## 事实核查

`fact_check_kol_claims()` 在 `fund_tools.py` 中，自动验证3类断言：

| 验证类型 | 模式 | 数据源 |
|---------|------|--------|
| 指数/板块涨跌 | `科创50跌3.5%` | `quotes` / `sectors` 的 `change_pct` |
| 成交额 | `成交2.6万亿` | `market_overview.total_turnover` |
| 北向资金 | `北向流出40亿` | `northbound.total` |

偏差等级: diff<0.3% → ✅, diff<1.0% → ⚠️, diff≥1.0% → ❌

结果追加到 `p['interpretation']`，格式:
```
📊 **事实核查**
> ✅ 科创50: 博主说跌3.5%, 实际-3.44%, 一致 ✓
```

## 评论分析（低权重）

仅对唐史主任的今日信号帖拉15条评论，只提取有主任回复的（最多3条），不做板块讨论分析。
