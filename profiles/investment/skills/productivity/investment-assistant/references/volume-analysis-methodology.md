# 量价分析系统文档

创建于 2026-07-06，因用户指正"分析只看涨跌幅不够"而建立。

## 核心哲学

价格反映"发生了什么"，成交量+振幅反映"发生的可靠性"。

- 放量上涨 → 趋势可靠，主力资金认可
- 无量空涨 → 可能诱多，谨慎追
- 放量下跌 → 资金出逃确认，规避
- 缩量阴跌 → 卖压减弱但买盘不足，等待

## 函数 API

```python
from fund_tools import get_volume_analysis

va = get_volume_analysis(quotes, sectors, prev_total_turnover=31818)
```

### 参数
- `quotes`: `get_all_quotes()` 返回的 dict，每个 value 需含 price/change_pct/high/low/prev_close/volume/turnover
- `sectors`: `get_sector_quotes()` 返回的 dict（2026-07-06 后支持 high/low/volume/turnover）
- `prev_total_turnover`: 前日两市总成交额（亿），用于判断放量/缩量

## 量价信号分类规则

| 条件 | 信号 | Emoji | 含义 |
|:----|:----|:-----:|:-----|
| pct > 1.5 AND amplitude > 3 | 放量上攻 | 🔥 | 典型资金进入，趋势可靠 |
| pct > 0.5 AND amplitude > 1.5 | 温和放量 | 📈 | 小幅放量上涨 |
| pct < -1.0 AND amplitude > 3 | 放量下跌 | 💧 | 资金出逃，规避 |
| pct < -0.5 AND amplitude > 2 | 放量回调 | 📉 | 放量但跌幅不大，性质中性偏弱 |
| abs(pct) < 0.3 AND amplitude < 1.5 | 缩量横盘 | ➖ | 多空平衡，等方向 |
| pct > 0 AND amplitude > 2 | 放量反弹 | 🔥 | 反弹有量 |
| pct < 0 AND amplitude < 1.5 | 缩量阴跌 | 💧 | 弱势但卖压不大 |

## 已知限制

1. **旧快照 sector 振幅=0** — 2026-07-06 之前的 `_yesterday_snapshot.json` 中 sectors 不含 high/low 字段，量价分析会跳过这些品种。需要等新快照覆盖。
2. **总成交额仅沪市** — `get_volume_analysis()` 从 quotes dict 中提取上证指数成交额，未包含深证（因 QUOTES 不含深证成指）。实际总成交额通过 `get_market_overview()` 获取，不影响各品种的独立量价信号。
