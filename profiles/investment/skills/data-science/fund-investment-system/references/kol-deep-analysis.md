# KOL 深度分析引擎

> **⚠️ 2026-07-15 更新：`generate_kol_deep_analysis()` 已从每日流水线中移除**
> 
> 用户要求不再对每条微博做黑话破译。现在仅对**今日信号帖**做选择性解读（`interpret_weibo()`），不再生成 `_kol_deep_analysis.txt`。

> 本参考保留字典内容供 `interpret_weibo()` 和手动分析使用。新方法论见 SKILL.md 中 **KOL 分析方法论** 章节及 `references/fact-check-kol-claims.md`。

## 概述

2026-07-14 新增：从关键词匹配升级为全域深度分析系统。核心文件 `fund_tools.py` 包含三大组件。

## 组件1：增强黑话字典（80+条，分5级）

| 层级 | 字典名 | 覆盖范围 | 示例 |
|:----|:-------|:---------|:-----|
| 一级：参与主体 | `SIGNALS_PARTICIPANTS` | 市场参与者/机构代称 | 村里=监管层, 柚子=游资, 汪汪队=国家队 |
| 二级：市场状态 | `SIGNALS_STATE` | 走势/波动描述 | ICU=暴跌底部, KTV=暴涨, 开线=暴跌回调 |
| 三级：交易行为 | `SIGNALS_ACTION` | 买卖操作代称 | 刷大火箭=重仓追高, 建仓=买入, 砸盘=卖出打压 |
| 四级：盈利风险 | `SIGNALS_PROFIT` | 盈亏/风险状态 | 吃肉=有盈利空间, 埋单=买入被套 |
| 五级：特定事件 | `SIGNALS_EVENTS` | 人名/事件/标的 | 长鑫=CXMT, 懂王=特朗普, bom交易=泡沫股 |

## 组件2：赛道→方向分类

`SECTOR_KEYWORDS` 常量（`fund_tools.py` 顶层）：
- 科技/AI: AI, 芯片, 半导体, 算力, 大模型, 机器人, GPU...
- 黄金: 黄金, 金价, 贵金属, 金矿...
- 资源/周期: 铜, 铝, 锂, 稀土, 煤炭, 钢铁...
- 新能源: 光伏, 锂电, 电池, 储能, 新能源车...
- 医药: 医药, 医疗, 创新药, CXO...
- 消费: 消费, 白酒, 食品, 家电...
- 市场整体: 大盘, 指数, 行情, A股, 创业板...

方向关键词（`_DIRECTION_BULLISH` / `_DIRECTION_BEARISH`）：
- 看多：加仓, 补仓, 右侧, 底部, 触底, 反弹, 抄底, 建仓, 吃肉
- 看空：减仓, 清仓, 风险, 警惕, 回调, 泡沫, 过热, 出货, 砸盘

## 翻译支持

`translate_text(text, target="zh-CN")` — 通过 Google Translate 免费 API 翻译
`is_mostly_english(text)` — 判定文本是否以英文字符为主（>40%）
- 用于 `format_rss_news()`，自动翻译英文标题为中文
- 带内存缓存，避免重复请求
- timeout=8s，失败回退显示原文
