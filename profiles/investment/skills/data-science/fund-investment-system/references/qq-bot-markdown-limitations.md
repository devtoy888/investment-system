# QQ Bot Markdown 限制与格式化规则

> 2026-07-19 用户多次纠正总结。QQ Bot 的 `msg_type=2` (markdown) 支持非常有限，与标准 Markdown 差异大。

## 不支持的标准 Markdown 语法

| 语法 | QQ Bot 渲染效果 | 替代方案 |
|:-----|:---------------|:---------|
| `## H2 标题` | **超大号字**（不可接受） | `📈 标题` emoji + 纯文本 |
| `**粗体**` | **大号字**（偏大） | 去掉 `**`，只用文字 |
| `#标签` | **H1 超大号字**（不可接受） | `标签` 去掉 `#` 前缀 |
| `_斜体_` | 斜体渲染异常 | `[括号]` 替代 |
| `- 列表项` | 部分支持，间距偏大 | 用空格缩进代替 |
| `> 引用` | 不支持 | 用缩进文字代替 |
| 连续空行 `\n\n\n` | 超大的段落间距 | 最多一个 `\n\n` |

## QQ Bot 安全格式（已验证可用）

```text
📈 本周大盘走势
| 指数 | 周涨跌 |
|:----|:----:|
| 科创50 | 🟢-7.12% |

📋 操作建议
  📈 增持 科技/AI 置信度53%
    涉及: 011613(5%)

📊 分析统计: 33条信号, 20条含方向
```

**规则：**
1. 表格 OK（`|` 分隔符、`:---:` 对齐）
2. emoji OK
3. 纯文本 OK
4. **所有 `**bold**` 必须删除**
5. **所有 `##` / `#` 必须删除**
6. 连续段落用 `\n\n` 分隔，不要多个空行
7. 段落内的折行用单个 `\n`

## 常见错误与修复（2026-07-19 实例）

| 推送 | 错误 | 修复 |
|:-----|:-----|:------|
| 周度复盘S1 | `## 📈 本周大盘走势` → 大号字 | `📈 本周大盘走势` |
| 周度复盘S2 | `**关键指数本周涨跌**` → 大号字 | `📊 关键指数本周涨跌` |
| 周度复盘S4 | `#ICU #KTV` → 超大号H1标题 | `ICU KTV` 纯文本 |
| 周度复盘S4 | `_(📎 接上条)_` → 斜体异常 | `[📎 接上条]` |
| 全量推送 | `**{worst_ch:+.2f}%**` → 数字偏大 | `{worst_ch:+.2f}%` 纯数字 |

## 验证方法

推送前检查：
1. 全文搜索 `**` → 全部删除
2. 全文搜索 `## ` 或 `### ` → 全部删除
3. 全文搜索 `#` 开头 → 确保不是 `#标签`
4. 全文搜索 `\\n\\n\\n` → 压缩为 `\\n\\n`
5. 全文搜索 `_text_` → 替换为 `[text]`
