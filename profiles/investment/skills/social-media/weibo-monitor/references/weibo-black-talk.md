## 三、唐史主任的说话模式

### 他的博文结构三要素

```
1. 事实陈述（可验证） — "美光的业绩出来了，全超"
   → 这是可验证的部分，独立于观点
   
2. 逻辑推导 — "所以存这条线没有问题了，泡沫论可以消退一段了"
   → 基于事实的推理链
   
3. 操作建议（可选）— "买ETF" / "开融资仓吸点"
   → 最直接的信号，但频率较低
```

### 信号可信度评估（基于验证结果）

| 信号类型 | 可信度 | 已验证次数 |
|---------|:------:|:---------:|
| 具体公司业绩预判 | ✅ 100% | 3/3（美光、存储等） |
| 市场数据引用 | ✅ 100% | 2/2（成交量3.76万亿精确匹配） |
| 大盘趋势判断 | ✅ 100% | 3/3（指数回踩、科创新高等） |
| 操作建议（加仓/买入） | 🟢 高 | 已验证建议方向正确 |
| 风险警示 | 🟡 中 | 模糊，需结合小浣熊 |

### 他的核心投资框架（理解他的底层逻辑）

```
硅基通胀（AI算力需求驱动）→ 存储芯片涨价（美光等业绩暴增）→ 
国产替代（长鑫等扩产）→ 供应链配套（伴娘公司受益）
     ↓
最终传递到指数（含科量提升）
```

只要这个框架不被打破，他就持续看多科技/半导体。框架被打破的信号：存储价格下跌、美光业绩miss、AI资本开支下降。

## 四、独特技术术语详解

### 「玻璃基」到底指什么？

他专门写过长文区分：
1. **消费电子级玻璃基板** — 面板/LCD用，不是他关心的
2. **AI场景下玻璃基封装载板** — 他说的就是这个。特点是"更多洞、更多层、更大片"
3. **Corning GlassBridge** — 玻璃光学桥接器，用玻璃做半导体级别连接器，完成"光纤→PIC光路桥接"

**投资方向**：光通信/光纤/光模块产业链上游。康宁的GlassBridge本质是解决AI数据中心的带宽瓶颈。

### 「长鑫伴娘」是什么？

他在6月24日说过："私下做了长鑫伴娘组合，最近走的非常好。不是以往的'概念股'，而是实锤fab合作的，近期都没有什么回调。"

意思是：他把长鑫存储（CXMT）的真实供应链厂商做成了一个投资组合。这些厂商有实际合作（实锤fab），不是炒概念。他认为这些小厂会先涨价，拉动大fab估值。

### 「硅基通胀」什么意思？

他的2026年核心叙事：
- AI吞噬大量产能
- 硅基芯片/服务器价格全面上涨
- "苹果涨价、XBOX涨价，这些涨价背后，都是AI吞噬了大量产能，硅基通胀传导才开始"
- 这意味着：涨价逻辑还没有走完，科技股仍有上涨空间
