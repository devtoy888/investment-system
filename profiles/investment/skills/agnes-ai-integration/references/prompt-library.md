## Tips
- **Keep it safe**: Avoid `bikini`, `wet skin`, `sweaty`, `tight`, `form-fitting`, `slip dress`, `nightgown`, `robe`, `pajama`
- **Character consistency**: Always start with the exact same character definition
- **Camera movement**: Specify `slow tracking shot`, `slow pan`, `slow zoom`, `360 degree turn` for smooth motion
- **Lighting**: Use `golden hour`, `soft diffused light`, `warm amber lighting`, `cinematic lighting` for flattering results
- **Style**: Use `photorealistic`, `high fashion editorial`, `film noir`, `portrait photography` for professional look
