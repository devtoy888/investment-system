## 一、核心目标

**帮助用户在每日30秒内做出基金操作判断：买/卖/持有/不动，以及"是否应该关注一个新赛道"。**

次要目标：
1. 对抗确认偏误（behavioral finance）
2. 积累投资判断的历史准确率
3. 发现未持仓但有潜力的板块/赛道

## 二、理论框架（引用可验证的金融策略）

### 2.1 投行晨会 3-Box 模型 — 决定推送结构

源自 Goldman Sachs / Morgan Stanley 每日盘前晨会的标准结构：

| Box | 问题 | 对应推送内容 |
|-----|------|------------|
| Box 1 | 隔夜发生了什么？ | 美股/外盘黄金/美元/事件 |
| Box 2 | 对我们的持仓有什么影响？ | 传导链分析+板块受影响程度 |
| Box 3 | 今天怎么操作？ | 买/卖/持有/调仓/建仓建议 |

**参考文献**：T. Koller, M. Goedhart, D. Wessels (McKinsey), "Valuation: Measuring and Managing the Value of Companies", 7th ed., Chapter on Investment Decision Frameworks.

### 2.2 因子动量 + 行业轮动 — 决定追踪什么数据

源自 Jegadeesh & Titman (1993) "Returns to Buying Winners and Selling Losers" (Journal of Finance)：

- **短期动量（3-12个月）**：A股存在正向动量效应，尤其科技/主题型基金
- **A股短期反转（1-4周）**：A股反转效应比美股更强（文献：《中国A股市场动量与反转效应研究》，上交所）
- **行业轮动**：资金在申万一级行业间流动，通常持续2-4个月

**对推送的指导**：
- 需要追踪申万一级行业本周表现、本月表现
- 需要追踪北向资金流向
- 当某行业的动量信号连续2周为正，且用户未持有该行业基金 → 发出"新赛道机会"信号

### 2.3 行为金融学 — 决定验证机制

Kahneman (2011) "Thinking, Fast and Slow"：

- **确认偏误**：人天然记住正确判断、忘记错误判断 → 需要系统性的收盘复盘记录准确率
- **处置效应（Shefrin & Statman, 1985）**：过早卖出盈利、过久持有亏损 → 推送应包含"持仓偏离度提醒"

### 2.4 Portfolio Rebalancing — 决定调仓触发条件

当某类资产实际占比偏离目标占比超过阈值时触发调仓建议：

| 分组 | 目标占比 | 触发阈值 |
|------|---------|---------|
| 科技（科创/半导体/科技/ESG） | 40% | ±5% |
| 黄金 | 20% | ±5% |
| 其他（平衡型） | 30% | ±5% |
| 现金/活期 | 10% | — |

## 四、博主画像方法论（核心创新）

### 4.1 方法论基础

采用 **资料饱和度理论（Data Saturation, Guest et al. 2006, Field Methods）**：
- 在定性分析中，当新数据不再带来新的主题/模式时，样本达到"饱和度"
- 应用于KOL分析：当增加一档新博文不再显著改变对KOL风格的认知 → 停止扩展

### 4.2 逐步采集迭代流程

```
Phase 0: 基线采集
  ├─ Pull 最近 20 条 per KOL
  ├─ 生成初始画像草稿
  └─ 评估：有信号价值的博文占比 > 20%？
      ├─ 是 → 下一步
      └─ 否 → 需要更多样本确认（噪音太多或近期没发）

Phase 1: 扩展至信号密度稳定
  ├─ Pull 下一个 30 条（累计 50 条 per KOL）
  ├─ 重新分析，对比 Phase 0 画像的一致性
  ├─ 一致性指标：前50条的信号词频率 vs 后20条
  │   - 差异 > 30% → 样本仍不稳定，继续扩展
  │   - 差异 < 15% → 达到初步饱和，进入 Phase 2
  └─ 不论结果，记录每次迭代的画像变化到 R2

Phase 2: 验证画像稳定性
  ├─ Pull 下一个 50 条（累计 100 条 per KOL）
  ├─ 使用 Phase 1 建立的信号词/风格模型
  │   对 Phase 2 的新博文做预判（此博主会如何说）
  ├─ 预判准确率 > 60% → 饱和，停止扩展
  └─ 预判准确率 ≤ 60% → 继续扩展到 150 条

Phase 3: 最终确认（如果需要）
  ├─ Pull 下一个 50 条（累计 150 条 per KOL）
  └─ 验证预判准确率是否超过 60%
      - 是 → 饱和，形成最终画像
      - 否 → 该博主可能无稳定风格，标记为"高噪声"来源
```

### 4.4 饱和度判断标准（可量化）

| 指标 | 公式 | 饱和阈值 |
|------|------|---------|
| 信号词稳定性 | 后N条的新词率 | < 15% |
| 风格分类稳定性 | 分类结果是否变化 | 连续2次一致 |
| 预判准确率 | Phase 1模型→Phase 2博文的判断正确率 | > 60% |
| 主题覆盖率 | 该博主覆盖的赛道数 | 连续2次不变 |

**停止条件：** 以上四个指标中至少三个达到阈值。

## 五、"发现新赛道"机制（你提出的第1点）

这是本次新增的核心功能。逻辑链：

```
触发条件（每天早上检查）：
1. 扫描申万一级行业指数（或热门ETF）的近期表现
2. 哪些行业过去5/10/20个交易日动量最强且你未持有？
3. 有KOL在最近博文中提及该行业吗？
4. 该行业与已有持仓的相关性如何？（避免内部对冲）

决策树：
  动量强 + KOL提及 + 与低相关性 → 强烈建议关注
  动量强 + 无KOL提及             → 温和建议观察
  动量强 + KOL提及 + 高相关性     → 已有持仓覆盖，不必重复
```

你未持有的赛道识别：
- 你的持仓覆盖了：科技（半导体/科创/ESG/恒生科技）、黄金、新能源、创新药、资源、通航
- **未覆盖的赛道示例**：消费/白酒、军工、煤炭/能源、银行、房地产、医疗健康、AI应用/软件

例如：如果军工板块连续2周领涨，且唐史主任多次提军工，但你完全没有军工基金 → 推送提示关注。
