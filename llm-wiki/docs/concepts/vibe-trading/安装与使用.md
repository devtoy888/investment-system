# 安装与基础使用

## 快速安装

### PyPI（推荐）

```bash
pip install vibe-trading-ai
vibe-trading init
vibe-trading
```

### UV 工具安装

```bash
uv tool install vibe-trading-ai
vibe-trading init
vibe-trading serve --port 8899
```

### 源码安装

```bash
git clone https://github.com/HKUDS/Vibe-Trading
cd Vibe-Trading
pip install -e .
vibe-trading
```

### Docker

```bash
docker compose up -d
# Docker 数据在重启后保留
```

## CLI 命令参考

| 命令 | 说明 |
|------|------|
| `vibe-trading run -p "..."` | 自然语言金融研究任务 |
| `vibe-trading init` | 初始化工作空间 |
| `vibe-trading serve --port 8899` | 启动 Web UI |
| `vibe-trading`（无参数） | 启动交互式 TUI |
| `vibe-trading --swarm-run <preset>` | 多智能体团队 |
| `vibe-trading --upload <file>` | 上传文件（CSV/PDF） |
| `vibe-trading alpha list` | 列出因子 |
| `vibe-trading alpha bench` | 评测因子 |
| `vibe-trading alpha show` | 查看因子详情 |
| `vibe-trading alpha compare` | 比较因子表现 |
| `vibe-trading channels status/start/stop` | IM 频道管理 |
| `vibe-trading connector list/use/check` | 券商连接器管理 |
| `vibe-trading data mode free/paid` | 切换数据模式（免费/QVeris） |
| `vibe-trading provider doctor` | 提供商诊断 |
| `vibe-trading setup` | 跨平台开发环境设置 |
| `vibe-trading-mcp` | 启动 MCP 服务器 (stdio) |
| `vibe-trading-mcp --transport sse` | 启动 MCP SSE 模式 |

### 使用示例

```bash
# 策略回测
vibe-trading run -p "Backtest BTC-USDT 20/50 MA for 2024, summarize return and drawdown"

# 市场研究
vibe-trading run -p "Research current market sentiment on AI chip stocks"

# 跨会话记忆
vibe-trading run -p "Remember: I prefer RSI-based strategies, max 10% drawdown"
vibe-trading run -p "Build a crypto strategy that fits my risk profile"

# 文档分析
vibe-trading --upload NVDA_Q1_earnings.pdf
vibe-trading run -p "Summarize the key risks from this earnings report"
```

## API 服务器

```bash
vibe-trading serve --port 8899
# 交互文档: http://localhost:8899/docs
```

### 核心 API 端点

| 方法 | 端点 | 说明 |
|------|------|------|
| `GET` | `/runs` | 列出运行记录 |
| `GET` | `/runs/{run_id}` | 运行详情 |
| `GET` | `/runs/{run_id}/pine` | 多平台指标导出 |
| `POST` | `/sessions` | 创建会话 |
| `POST` | `/sessions/{id}/messages` | 发送消息 |
| `GET` | `/sessions/{id}/events` | SSE 事件流 |
| `POST` | `/upload` | 上传 PDF/文件 |
| `GET` | `/swarm/presets` | 列出 Swarm 预设 |
| `POST` | `/swarm/runs` | 启动 Swarm |
| `GET` | `/swarm/runs/{id}/events` | Swarm SSE |
| `GET` | `/alpha/list` | 列出 Alpha 因子 |
| `POST` | `/alpha/bench` | 评测 Alpha |
| `GET` | `/settings/llm` | 读取 LLM 设置 |
| `PUT` | `/settings/llm` | 更新 LLM 设置 |
| `GET` | `/settings/data-sources` | 数据源设置 |
| `PUT` | `/settings/data-sources` | 更新数据源设置 |
| `GET` | `/channels/status` | IM 频道状态 |
| `POST` | `/channels/start/stop` | 启动/停止频道 |
| `POST` | `/scheduled-runs` | 创建定时研究任务 |

### 安全配置

| 环境变量 | 说明 |
|----------|------|
| `API_AUTH_KEY` | API 认证密钥（非 localhost 需要） |
| `VIBE_TRADING_ENABLE_SHELL_TOOLS` | 启用 Shell 工具（默认关闭） |
| `VIBE_TRADING_ALLOWED_FILE_ROOTS` | 允许的文件读取根目录 |
| `VIBE_TRADING_ENABLE_SCHEDULER` | 启用定时任务调度器 |
| `TUSHARE_TOKEN` | Tushare 数据 Token |
| `FMP_API_KEY` | Financial Modeling Prep 密钥 |

### 安全默认值
- Shell 工具仅限本地 CLI 和受信 localhost；远程 API 会话默认不暴露
- 文件/日志读取器限制在 `agent/uploads`、`agent/runs`、`./uploads`、`./data`、`~/.vibe-trading/uploads`、`~/.vibe-trading/imports` 等根目录
- 回测子进程环境严格限制：只透传 OS/Python 基础环境、代理/证书设置、市场数据只读密钥，不透传 LLM 密钥和券商交易密钥

## Web UI 设置页面
- 支持在线配置 LLM 提供商/模型/Base URL/生成参数/reasoning effort
- 支持配置市场数据凭证（如 Tushare Token）
- 设置写入 `agent/.env`
- IM 频道面板：查看状态、启动/停止适配器
