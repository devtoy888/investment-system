# Vibe-Trading 技术架构

## 目录结构

```
Vibe-Trading/
├── agent/                          # 后端 (Python)
│   ├── cli/                        # CLI 包 — 交互式 TUI + 子命令
│   ├── api_server.py               # FastAPI 服务器
│   ├── mcp_server.py               # MCP 服务器 (54 个工具)
│   │
│   ├── src/
│   │   ├── agent/                  # ReAct Agent 核心
│   │   │   ├── loop.py            #   5 层压缩 + 读写工具批处理
│   │   │   ├── context.py         #   系统提示 + 自动回忆持久记忆
│   │   │   ├── skills.py          #   技能加载器 (86 内置 + 用户创建)
│   │   │   ├── tools.py           #   工具基类 + 注册表
│   │   │   ├── memory.py          #   每轮运行的工作空间状态
│   │   │   └── trace.py           #   执行追踪写入器
│   │   │
│   │   ├── memory/                # 跨会话持久记忆
│   │   ├── tools/                 # 68 个自动发现 Agent 工具
│   │   ├── factors/               # Alpha Zoo — 460 因子
│   │   ├── api/                   # FastAPI 路由模块
│   │   ├── skills/                # 86 个金融技能 (9 类)
│   │   ├── swarm/                 # Swarm DAG 执行引擎
│   │   └── providers/             # LLM 提供商抽象层
│   │
│   └── backtest/                  # 回测引擎
│       ├── engines/               #   7 引擎 + 组合跨市场引擎
│       ├── loaders/               #   20 个数据源加载器
│       └── optimizers/            #   投资组合优化器 (MVO/等权/最大分散/风险平价)
│
├── frontend/                      # Web UI (React 19 + Vite + TypeScript)
└── Dockerfile                     # 多阶段构建
```

## Agent 核心 (ReAct Loop)

### 5 层上下文压缩 (`loop.py`)
1. **工具结果回顾** — 只在 token 压力下清理旧工具结果
2. **会话历史摘要** — 长时间对话自动摘要
3. **技能上下文** — 按需加载技能 SKILL.md
4. **记忆注入** — 持久记忆自动注入系统提示
5. **核心系统提示** — 角色定义 + 安全边界

### 68 个 Agent 工具
涵盖：回测运行、跨会话记忆（保存/回忆/遗忘）、技能 CRUD、FTS5 会话搜索、Swarm 启动、网页搜索、bash 执行、文件 I/O、因子分析、期权分析、Alpha 浏览器/评测等。

### LLM 提供商抽象 (`src/providers/`)
支持多个 LLM 提供商，通过 `llm_providers.json` 配置。关键功能：
- 提供商预检（健康检查）
- 内容过滤弹性（跳过个别 LLM 内容审核命中）
- 推理流（GLM、Gemini 等）

## 会话系统 (`src/session/`)
- 多轮对话支持
- FTS5 全文搜索
- 会话 `resume` 恢复
- 持久化到 `~/.vibe-trading/memory/`

## 技能系统 (`src/skills/`)
- 86 个内置金融技能，9 个类别
- 每个技能是一个 SKILL.md 文件
- 用户可通过 Agent 创建/修改/删除
- 技能发布到 OpenSpace 平台可自进化

## 安全边界

| 安全机制 | 说明 |
|----------|------|
| Shell 工具默认关闭 | `VIBE_TRADING_ENABLE_SHELL_TOOLS=1` 才开启 |
| 文件操作隔离 | 限制在 `uploads/`, `runs/`, `data/` 等指定目录 |
| API 认证 | 非 localhost 需要 `API_AUTH_KEY` |
| SSRF 防护 | 阻止 CGNAT/内网目标 |
| 回测子进程隔离 | 仅透传市场数据密钥，不透传 LLM 密钥 |
| 内容过滤弹性 | Swarm 跳过 LLM 内容审核命中，记录警告 |
