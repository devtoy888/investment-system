# 数据源与回测引擎

## 19 个免费市场数据源

Vibe-Trading 内置 19 个免费数据源，按 IP 封禁风险智能回退。

### A 股数据源回退链

| 层级 | 源 | 认证 | 特点 |
|------|-----|------|------|
| 1 | `tencent` · `mootdx` | 无 | **永不封 IP**，优先使用 |
| 2 | `eastmoney` | 无 | OHLCV + 基本面 |
| 3 | `baostock` · `akshare` | 无 | 免费回退 |
| 4 | `tushare` | Token | 最丰富 A 股数据（需注册） |

### 美股数据源回退链
`yahoo` → `stooq` → `sina` → `eastmoney` → `yfinance`

### 港股数据源回退链
`eastmoney` → `yahoo` → `yfinance`

### 加密货币数据源回退链
`okx` → `ccxt` (100+ 交易所) → `yfinance`

### 印度市场（v0.1.11 新增）
- 专用 `IndiaEquityEngine`：T+1 交割、涨跌停板、STT/印花税/交易所/SEBI/GST 费用模型
- `.NS` (NSE) / `.BO` (BSE) 符号路由
- 可选只读 Shoonya/Dhan 数据桥

### 可选付费数据
- **QVeris**：63+ 提供商，通过 Settings → QVeris 或 `vibe-trading data mode paid` 启用

### 专用加载器列表（20 个）
tushare, okx, yfinance, akshare, baostock, tencent, mootdx, ccxt, futu, local, eastmoney, sina, stooq, yahoo, finnhub, alphavantage, tiingo, fmp, qveris, india_broker

## 7 大回测引擎

| 引擎 | 覆盖市场 |
|------|---------|
| 股票引擎 | A 股、港股、美股、印度 |
| 加密引擎 | BTC-USDT 等 |
| 期货引擎 | 期货市场 |
| 外汇引擎 | 外汇市场 |
| 组合引擎 | 跨市场组合 |
| 期权组合引擎 | 期权组合策略 |
| 印度股票引擎 (v0.1.11) | NSE/BSE |

### 回测特性
- PIT 数据对齐（Point-in-Time 防未来信息泄漏）
- 生成策略代码 + PineScript/TDX/MT5 导出
- Run Card 格式：含回测指标、工具追踪、引用的可复现报告
- `validation.json`：严格 JSON 格式（NaN/Infinity 规范化）

## 投资组合优化器

| 优化器 | 类型 |
|--------|------|
| MVO | 均值-方差优化 |
| Equal Vol | 等波动率 |
| Max Div | 最大分散化 |
| Risk Parity | 风险平价 |
| Turnover-aware (v0.1.11) | 考虑换手率的优化器 |

## 数据加载器架构

所有加载器遵循 `DataLoader Protocol`：
- 统一的 `fetch()` 接口
- 注册表自动管理回退链
- 按市场（A/US/HK/Crypto）自动选择回退顺序
- 加载器缓存隔离（Windows 兼容）
