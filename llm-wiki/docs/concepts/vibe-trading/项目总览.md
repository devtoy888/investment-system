# Vibe-Trading 项目总览

## 是什么

**Vibe-Trading** 是香港大学数据科学实验室（HKUDS）开源的**金融研究 Agent 工作台**。它将自然语言问题转化为可运行的金融分析流程——连接市场数据加载、策略生成、回测引擎、文档分析、交易日志诊断、持久化记忆和多智能体研究团队。

> 项目定位：**Research, Simulation, Backtesting**——不做投资建议，不持有资金，不运行执行场所。

## 四大核心能力

### 🔍 自改进交易 Agent（Agent Harness）
- 自然语言市场研究
- 跨会话持久化记忆
- 会话级 FTS5 搜索
- 86 个可编辑金融技能
- 5 层上下文压缩

### 📊 跨市场数据与回测
- **7 大回测引擎**：覆盖股票、加密货币、期货、外汇、组合、期权组合
- **19 个免费数据源**：支持 A 股/港股/美股/加密/期货/外汇
- 智能回退链：按 IP 封禁风险自动切换数据源
- 可选的 QVeris 付费数据（63+ 供应商）

### 🐝 多智能体交易团队（Swarm）
- 投资委员会、量化团队、风控团队、宏观团队、加密团队等 **30 个预设**
- DAG 执行引擎，流式进度报告
- Worker 用实时市场数据作为依据

### 👥 Shadow Account（影子账户）
- 解析券商交易日志
- 提取隐式交易规则
- 运行反事实回测
- 生成审计报告

## 生态项目

Vibe-Trading 是 HKUDS Agent 生态的一部分：
- **NanoBot** — 超轻量个人 AI 助手
- **AI-Trader** — Agent 原生信号与跟单平台
- **CLI-Anything** — 让所有软件 Agent 原生
- **OpenSpace** — 自进化 AI Agent 技能
- **ClawTeam** — Agent Swarm 智能
